# 自助银行

## 项目概述
开发一个现代化、安全可靠的**智能银行全流程业务系统**，采用**SpringBoot 3.2 + Vue3 + TypeScript**技术栈，实现从客户开户到复杂金融交易的全业务闭环，满足银行级安全与性能要求，界面设计需体现金融专业性与现代化体验。

## 核心业务要求

### 1. **客户全生命周期管理**
- **智能开户**：OCR识别证件+活体检测，自动填充客户信息
- **KYC认证**：风险等级评估，合规验证流程
- **客户画像**：基于交易行为生成360°客户视图
- **客户分级**：VIP/普通客户差异化服务策略

### 2. **账户体系**
- **多账户类型**：
  - 个人储蓄账户（活期/定期）
  - 企业结算账户
  - 信用卡账户
  - 投资账户
- **账户全景**：余额实时计算，资金流向可视化
- **智能提醒**：大额变动、余额不足、定期到期自动提醒

### 3. **交易核心**
- **实时交易**：
  - 同行/跨行转账（支持批量）
  - 扫码支付/收款
  - 存取款（与ATM模拟集成）
- **交易流水**：
  - 多维度查询（时间/类型/对手方）
  - 电子回单自动生成
  - 交易状态实时追踪
- **交易风控**：
  - 大额交易二次验证
  - 异地交易预警
  - 反洗钱规则引擎

### 4. **贷款管理**
- **贷款产品矩阵**：
  - 个人消费贷
  - 住房按揭
  - 企业经营贷
  - 信用贷
- **全流程管理**：
  - 智能额度测算
  - 自动化审批（规则引擎+AI辅助）
  - 还款计划生成
  - 逾期管理

### 5. **财富管理**
- **理财产品超市**：
  - 基金/保险/信托产品
  - 风险等级匹配
  - 收益率历史数据可视化
- **投资组合**：
  - 资产配置建议
  - 收益风险分析
  - 智能调仓提醒

### 6. **运营支持**
- **柜员工作台**：一站式业务处理
- **审批中心**：多级审批流程
- **报表中心**：监管报表自动生成
- **监控大屏**：实时业务数据可视化

## 技术架构要求

### 前端架构 (Vue3 + TypeScript)
```markdown
- **框架**：Vue3 Composition API + TypeScript
- **状态管理**：Pinia (模块化设计)
- **路由**：Vue Router (动态路由+权限控制)
- **UI组件库**：基于Element Plus深度定制
- **可视化**：ECharts + D3.js (数据大屏)
- **构建工具**：Vite (生产环境优化)
- **代码规范**：ESLint + Prettier + Husky
```

### 后端架构 (SpringBoot 3.2)
```markdown
- **核心框架**：Spring Boot 3.2 + Spring Cloud Alibaba
- **持久层**：MyBatis-Plus + ShardingSphere (分库分表)
- **缓存**：Redis (多级缓存策略)
- **消息队列**：RocketMQ (异步解耦)
- **安全框架**：Spring Security + OAuth2.0 + JWT
- **文档**：SpringDoc OpenAPI 3.0
- **监控**：Prometheus + Grafana + SkyWalking
```

### 部署架构
- **容器化**：Docker + Kubernetes
- **服务网格**：Istio (流量管理)
- **CI/CD**：GitLab CI (多环境流水线)
- **数据库**：MySQL集群 (主从读写分离) + MongoDB (日志存储)

## 界面设计规范

### 设计风格
- **主色调**：深蓝色(#0A2463) + 金色(#D4AF37) (体现金融专业与尊贵感)
- **辅助色**：科技蓝(#1E88E5) + 安全绿(#43A047)
- **字体**：主标题使用PingFang SC Bold，正文使用HarmonyOS Sans
- **设计原则**：
  - **简洁专业**：避免过度装饰，突出金融数据
  - **一致性**：统一的组件库，色彩/间距/动效规范
  - **响应式**：适配桌面/平板/大屏显示

### 核心界面要求
1. **登录页**：
   - 深色背景+金色渐变，3D银行建筑模型
   - 多因素认证流程（密码+短信/生物识别）
   - 安全登录动画反馈

2. **仪表盘**：
   - 顶部：资金概览卡片（总资产、可用余额、投资收益）
   - 中部：交易热力图 + 实时交易流
   - 底部：快捷操作 + 重要提醒
   - 动态数据更新（WebSocket实时推送）

3. **转账页面**：
   - 三步流程：收款人→金额→确认
   - 收款人智能搜索（历史记录+联系人）
   - 金额输入大数字键盘
   - 交易安全盾（实时风险评估显示）

4. **贷款中心**：
   - 产品卡片式布局，利率/期限/额度清晰展示
   - 智能匹配算法（根据客户画像推荐）
   - 进度可视化（申请→审批→放款）

5. **财富管理**：
   - 资产分布环形图（现金/固收/权益/另类）
   - 收益曲线对比（基准/市场/个人）
   - 产品详情3D翻转展示

6. **管理后台**：
   - 柜员工作台：待办事项聚合
   - 审批中心：流程图可视化
   - 风控大屏：实时交易地图+异常监控

## 安全合规要求

### 数据安全
- **传输加密**：TLS 1.3 + 国密算法
- **存储加密**：敏感字段AES-256加密
- **脱敏规则**：账户号、身份证号自动脱敏
- **审计日志**：关键操作全记录（谁/何时/做了什么）

### 业务安全
- **交易风控**：
  - 实时反欺诈引擎
  - 交易限额动态管理
  - 设备指纹识别
- **合规要求**：
  - 满足《个人金融信息保护技术规范》
  - 支持GDPR数据删除权
  - 反洗钱(AML)规则集成

### 权限控制
- **RBAC模型**：角色-权限-数据权限三层控制
- **敏感操作**：关键交易强制审批
- **会话管理**：自动超时+异地登录提醒

## 性能要求
